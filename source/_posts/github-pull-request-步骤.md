title: github pull request 步骤
date: 2022-08-18 20:02:37
tags: [Github]
---

```bash
>> mkdir [filename]
## 製作一個檔案夾

>> cd [filename]

先在想要pull request的主題建立fork到自己的帳號，然後在clone
>> git clone https://github.com/YuWeiKuo/[主題]

>> cd [主題]

修改主題裡的檔案

>> git add [修改的檔案]
>> git commit -m "說明"
上傳到自己的fork
>> git push

到github裡自己的專案裡按New pull request

```
